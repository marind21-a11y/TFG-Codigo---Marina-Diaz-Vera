# Código del Trabajo de Fin de Grado

Este repositorio contiene el código completo utilizado para el Trabajo de Fin de Grado de Marina Díaz Vera.

## Contenido

- `CODIGO_TFG_FINAL.R`: script completo en R utilizado para la descarga de datos, construcción de variables, análisis exploratorio y estimación de los modelos econométricos.

## Fuentes de datos

El código utiliza datos financieros procedentes de Yahoo Finance y variables macroeconómicas del Instituto Nacional de Estadística.

## Modelos incluidos

El script incluye la construcción de variables y la estimación de modelos ARIMA/SARIMA, contrastes de causalidad de Granger, regresiones con dummies de crisis, modelos GARCH y contrastes de cointegración.
